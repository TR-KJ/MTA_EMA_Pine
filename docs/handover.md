これから渡すコードは、1分足チャート上に15分足MTAを表示するための土台インジケーターです。

## 土台コードの位置づけ

- 名称：v8.1α-A-MTA15 Base
- 目的：1分足チャート上に、15分足MTAゾーンと15分MTA対応高値/安値を表示する
- 役割：今後のEMAトリガー開発用のベースコード
- 重要：このコードにはEntry、SL/TP、Strategy要素、Pivot Break Triggerは含めない

## このコードで残しているもの

- 15分足MTA検出ロジック
- 15分MTAゾーン表示
- 15分MTA対応高値/安値表示
- 1分足上でのMTAゾーン接触判定
- MTAゾーン接触アラート

## このコードで削っているもの

- B1/B2/B5系のPivot Break Trigger
- Break H / Break L
- Update H / Update L
- Pre Break H / L
- Entry条件
- SL/TP
- strategy() 関連処理

## 絶対ルール

1. まず、私が次に貼る `v8.1α-A-MTA15 Base` のコードを読み込んでください。
2. 15分MTA検出ロジックは勝手に変更しないでください。
3. 15分MTAゾーン表示、対応高値/安値表示、MTA接触アラートの基本挙動も勝手に変更しないでください。
4. 今後追加・変更するのは、基本的に「1分足側のEMAトリガー条件」のみです。
5. まずはストラテジー化せず、インジケーターとしてEMAトリガーの挙動確認を行います。
6. EMAトリガーの挙動確認がOKになってから、Strategy化します。
7. コード変更時は、変更点を明確に説明してください。
8. Pine Scriptでは、1行1文を守り、セミコロンで複数文を書かないでください。
9. `request.security()` を使う場合は、必ず `lookahead = barmerge.lookahead_off`、`gaps = barmerge.gaps_off` を使ってください。
10. `pine_check` が通っても、TradingView実機でランタイムエラー確認が必要であることを前提にしてください。

## 今後やりたいこと

15分MTA Setupに対して、1分足側でEMAを使ったTriggerを検証・開発したいです。

イメージは以下です。

- 1分足チャートに表示された15分MTAゾーンに価格が接触
- その後、1分足側のEMA条件でTrigger判定
- まずはインジケーターでTrigger表示のみ
- Entry、SL、TPはまだ実装しない
- 挙動確認後にStrategy化してバックテストする

## EMAトリガー案で考えたい方向性

候補としては、以下のようなものを検討したいです。

- MTA Setup後、1分足の終値がEMAを実体上抜け/下抜けしたらTrigger
- MTA Setup後、短期EMAが長期EMAをクロスしたらTrigger
- MTA Setup後、価格がEMA上/下に乗り、かつEMAの傾きが有利方向ならTrigger
- MTA Setup後、EMAへの押し戻り確認後にTrigger
- EMAをTriggerではなく、フィルターとして使う

ただし、最初から複雑にせず、まずは挙動確認しやすいシンプルなEMAトリガー案から始めたいです。
